# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python and SQL to uncover purchasing patterns and trends. The insights help understand customer preferences and buying behavior.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook

## Key Highlights
- Performed end-to-end Exploratory Data Analysis (EDA) including data cleaning and preprocessing.
- Analyzed customer behavior patterns such as purchase frequency, product categories, and spending trends.
- Created meaningful visualizations and dashboards to present actionable insights.

## Files Included
- `Customer_Shopping_Behavior.ipynb` – EDA and analysis
- `customer_shopping_behavior.csv` – Dataset
- `Customer_behavior.sql` – SQL queries
- `customer_behavior.pbix` – Power BI dashboard

## Outcome
Identified key factors influencing customer purchasing behavior and presented insights using visual analytics.
